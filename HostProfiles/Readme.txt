Host Prfiles

Host Profiles is a cross platofrm host file management tool. Quickly switch host profiles and flush dns automatically. 

Requirement:

Windows: 
.Net framework 2.0 and upper.

Linux: 
Mono 1.2 and upper
nscd Service and ifconfig commmand

Linux help

Ubuntu 10.0 Tested

Currently, only root user can launch the program.
Please launch the program like this:
sudo mono HostProfiles.exe
